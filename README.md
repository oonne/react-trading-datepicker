# react-textarta
------------

Textarea  for self-adaptation height

### Demo
[Demo](https://oonne.github.io/react-textarea/build/)

### How to start
* install dependencies
> npm install

* run server
> npm run test

* browse 'localhost:3000'


### Description

``` javascript
import Textarta from './component/Textarta'

<Textarta
  value={value}

/>
